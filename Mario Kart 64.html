Welcome to Mario Kart

When you first enter the game, it will ask you to press start like the picture down below...

To start, press ENTER.

Once you get in, it will show you a game select screen, If it's just yourself playing, use the WASD keys to select option "One Player" and press K

Select the mode that you want, and press ok

Mario Kart Grand Prix

If you enter as one player, Mario will ask you to select your player.

Select any player and press k for OK.

Once you do that, it will take you the the Map select screen.

Select any map and press ok.
